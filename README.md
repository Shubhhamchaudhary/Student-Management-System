# Student Management System (C++)

##  Description
A simple console-based application to manage student records using C++ and file handling.

##  Features
- Add student
- View all students
- Search student by ID
- Delete student
- Data stored using file handling

##  Technologies Used
- C++

##  How to Run
1. Compile the code:
   g++ main.cpp -o main
2. Run the program:
   ./main
